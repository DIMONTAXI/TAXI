2. Порядок установки
    1.в командной строке выполнить
        sqlplus <Имя схемы>/<пароль схему>@<connect_string> @install.sql
         <Имя схемы> - имя схемы БД, куда устаналиваеся продукт TAXI
         <connect_string> -  строка TNS подключение к базе данных
    2. Проверить на налице ошибок  файл install.log
